# r_OddOrEven

Find if a given integer is Odd or Even.

Hint: Use(%) - modular operator.

% operator returns the remainder. Example: 15 % 4 will give you 3. (15/4 , quotient is 3 and remainder is 15-12 = 3 ).

## Input Format

Input will be an integer.

## Constraints

0< n <100.

## Output Format

Output: "Odd" or "Even".
